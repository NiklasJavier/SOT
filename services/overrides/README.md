# Services Overrides

Dieses Verzeichnis ist für umgebungsspezifische Konfigurationsdateien vorgesehen, die Standardwerte aus `services/default_config.yml` überschreiben sollen. Dateien in diesem Ordner werden vom Setup-Skript automatisch angelegt und können beispielsweise branch- oder kundenbezogene Variablen aufnehmen.
