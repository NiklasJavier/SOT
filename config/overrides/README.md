# Config Overrides

Dieses Verzeichnis ist für umgebungsspezifische Konfigurationsdateien vorgesehen, die Standardwerte aus `config/default_config.yml` überschreiben sollen. Dateien in diesem Ordner werden vom Bootstrap-Skript automatisch angelegt und können beispielsweise branch- oder kundenbezogene Variablen aufnehmen.
